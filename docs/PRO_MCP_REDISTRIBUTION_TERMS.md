# Pro API, MCP and redistribution terms

13FLOW Pro is not a public self-serve SaaS product. It is currently a
limited-capacity, operator-reviewed technical evaluation surface.

## Current operating boundary

- No public package pricing.
- No public self-serve checkout.
- No uptime SLA.
- No support SLA.
- No enterprise procurement promise.
- No production x402 paid access.
- No redistribution right without written approval.

Access can be declined, rate-limited, paused, rotated or revoked for operational,
security, abuse-prevention or legal reasons.

## Allowed pilot use

A scoped pilot key may be used for:

- internal research;
- notebooks and dashboards;
- data-quality checks;
- bounded API integration tests;
- MCP agent workflow evaluation where Pro tools fail closed without a valid key.

The buyer must accept that 13FLOW is a research screen over public SEC
EDGAR-derived filings, not investment advice.

## Restricted use

Do not allow without written terms:

- resale;
- redistribution;
- public embedding of Pro responses;
- bulk republishing;
- client-side API-key use;
- high-volume automated extraction beyond agreed limits;
- claims that 13FLOW provides validated alpha, probabilities, price targets or
  expected returns.

## Validation wording

Permitted wording:

> 13FLOW has a 25-ticker mature 13F + Form 4 joined evidence artifact that is
> mechanically schema-valid and ready for human methodology review.

Forbidden wording:

- validated alpha;
- proven signal;
- expected-return model;
- probability model;
- investment recommendation;
- complete insider universe;
- full out-of-sample validation.

## Redistribution

API-key access never implies redistribution rights. Redistribution, resale,
customer-facing embedding, custom limits, uptime commitments and procurement
terms require a separate written agreement before any production use.
